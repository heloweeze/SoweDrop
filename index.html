Guide Utilisateur SoweDrop

Ce guide d'accompagnement vous prend en main pas à pas pour vous aider à maîtriser l'ensemble des fonctionnalités de notre application.

Étape 1 : Installation et premier démarrage

  • Téléchargement : Téléchargez le fichier d'installation depuis notre site web officiel.
  • Installation : Double-cliquez sur le fichier pour lancer l'installation automatique sur votre système.
  • Ouverture : Lancez l'application SoweDrop. L'interface principale s'ouvre sur votre écran, prête à l'emploi.

Étape 2 : Découverte du tableau de bord initial

Lorsque vous vous connectez pour la première fois, votre tableau de bord central est totalement vide. Dès l'ouverture, vous pouvez valider les fenêtres d'accueil avec la touche Entrée pour accéder pleinement à l'espace.

Étape 4 : Prise en main des fonctionnalités clés

A. Ajouter et supprimer un document

  • Importer un document : Cliquez sur le menu « Importer ». Un formulaire guidé s'ouvre : renseignez le titre, l'auteur, les mots-clés ainsi que la description, puis sélectionnez le fichier sur votre ordinateur.
    💡 Raccourci clavier : Ctrl + O
  • Supprimer un document : Sélectionnez une ligne dans votre tableau et demandez la suppression. Grâce à notre système de corbeille intégrée, le document est masqué de vos recherches mais reste protégé contre les fausses manipulations.
    💡 Raccourci clavier : Ctrl + D

B. Rechercher efficacement avec la multicritères

La barre supérieure de l'application vous permet de filtrer très précisément vos documents mais aussi des petites fonctionnalités :

  • Recherche par texte : Filtrez instantanément par le Titre, l'Auteur ou par Catégories.
  • Recherche par date : Encadrez vos documents en définissant une Date minimale et même une Date maximale.
  • Système de tri : Organisez l'affichage en choisissant votre axe (Trier par Titre, Date ou Auteur) ainsi que le sens (Ordre Croissant ou Décroissant).
  • Bouton RECHERCHER : Applique l'ensemble de vos critères et affiche les documents correspondants.
  • Bouton RÉINITIALISER : Vide instantanément tous les filtres saisis pour vider l'affichage lourd et réafficher la totalité de vos documents dans un format visuel léger.
    💡 Raccourci clavier : Entrée
  • Bouton Synchronisation : Après avoir importé ou supprimé un document, la fonctionnalité synchronisée permet de coordonner les documents partagés.
    💡 Raccourci clavier : Ctrl + R

C. Consulter, ouvrir et exporter un document

Faites un simple clic sur un document dans le tableau pour interagir avec lui :

  • Ouvrir le document : Lance instantanément la lecture de votre fichier en utilisant l'application par défaut installée sur votre bureau (votre lecteur PDF habituel, Word, etc.).
  • Télécharger : Effectue une copie locale de sécurité du fichier physique directement dans le dossier de votre choix sur votre ordinateur.
  • Copier le chemin de la ressource : Permet de copier instantanément le chemin d'accès absolu du fichier dans votre presse-papiers.

D. Confort visuel et assistance

  • Préférences de couleurs : Adaptez l'application à la luminosité de votre bureau. Basculez instantanément entre le Mode Clair et le Mode Sombre.
    💡 Raccourci clavier : Ctrl + P
  • Onglet Contact & Support : Un espace dédié au sein de l'application contient les informations de support et le lien vers le site web officiel.
